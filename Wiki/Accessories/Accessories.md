# Accessories / Curios

Accessories and Curios are two mod systems that create additional equipment slots on your character, allowing you to equip items beyond the standard armor slots. These systems are essential for equipping items like elytra, backtanks, backpacks, necklaces, and various other accessories without sacrificing your armor or inventory space.

## About

The accessory system in SimplyFellas is built on two complementary mods: **Accessories** and **Curios API**. Together, they provide a flexible framework for additional equipment slots that mods can use to add wearable items. This system allows you to:

- Equip your elytra without removing your chestplate
- Wear backtanks and jetpacks from Create mods
- Use backpacks as wearable items
- Equip cosmetic armor pieces
- Wear necklaces, rings, and other accessories
- Use various utility items that benefit from being equipped rather than held

This system greatly enhances gameplay by allowing you to combine multiple utility and cosmetic items simultaneously, making character customization and optimization much more flexible.

## How It Works

### Accessories Mod

The Accessories mod provides a simple, lightweight accessory slot system. It's designed to be compatible with various mods and allows for easy addition of accessory slots.

### Curios API

Curios API is the more comprehensive system, providing a framework that other mods can use to add their own accessory slots. It supports multiple slot types and allows mods to define which items can go in which slots.

### Elytra Slot

The Elytra Slot mod adds a dedicated back slot specifically for elytra. This allows you to equip your elytra without using your chestplate slot, so you can fly while still wearing full armor!

### Create Integration

Create mods integrate with the accessory system to allow you to equip Create-specific items like backtanks and jetpacks from Create: Stuff & Additions in accessory slots, making them more convenient to use.

## Getting Started

### Opening the Accessory Menu

To access your accessory slots:

1. **Press the Inventory Key**: Default is `E` (or your configured inventory key)
2. **Locate Accessory Slots**: Look for additional equipment slots around your character model
3. **Identify Slot Types**: Different slots are labeled (Back, Necklace, Ring, etc.)
4. **Drag and Drop**: Simply drag items from your inventory into the appropriate slot

### Available Slot Types

The accessory system provides several slot types:

- **Back Slot**: For elytra, backpacks, backtanks, and jetpacks
- **Necklace Slot**: For necklaces and amulets
- **Ring Slots**: Multiple ring slots for various ring types
- **Belt Slot**: For utility belts and pouches
- **Charm Slot**: For charm items and trinkets

### First Accessory: Elytra

The most common first accessory is the elytra:

1. **Obtain an Elytra**: Find one in End Cities or trade with villagers
2. **Open Inventory**: Press `E` to open your inventory
3. **Locate Back Slot**: Find the back/elytra slot (usually above or to the side of your character)
4. **Equip Elytra**: Drag the elytra into the back slot
5. **Wear Armor**: You can now wear a chestplate in the normal armor slot!

## Detailed Usage Guides

### Elytra Management

**Equipping Elytra:**
- Drag elytra to the back slot in your inventory screen
- The elytra will appear on your character's back
- You can now wear a chestplate simultaneously for full protection

**Elytra Functionality:**
- Works exactly like vanilla elytra when equipped in the back slot
- Can be enchanted with Mending, Unbreaking, etc.
- Fireworks still work for boosting while flying
- Repair with phantom membranes at an anvil

**Tips:**
- Keep a spare elytra in your ender chest for emergencies
- Use Mending enchantment to keep it repaired automatically
- Combine with Create jetpacks for enhanced flight capabilities

### Create Backtank Integration

**What are Backtanks?**
- Create backtanks store compressed air for powering Create tools
- Essential for using Create drills, saws, and other powered tools
- Can hold varying amounts of air depending on tier

**Equipping Backtanks:**
1. Craft a Create backtank (Copper, Iron, or Brass tier)
2. Fill it with compressed air using Create's air compression system
3. Equip it in the back accessory slot (via Curios integration)
4. Your Create tools will now draw air from the equipped backtank

**Backtank Tiers:**
- **Copper Backtank**: Basic tier, holds moderate air
- **Iron Backtank**: Improved tier, holds more air
- **Brass Backtank**: Advanced tier, holds the most air

**Usage Tips:**
- Keep multiple backtanks filled and ready to swap
- Use Create's spout system to automatically fill backtanks
- Combine with Create: Stuff & Additions for enhanced functionality

### Create Jetpack Integration

**What are Jetpacks?**
- Jetpacks from Create: Stuff & Additions provide flight capabilities
- Powered by compressed air (similar to backtanks)
- Allow for controlled vertical and horizontal movement

**Equipping Jetpacks:**
1. Craft a Create jetpack (requires Create: Stuff & Additions)
2. Fill with compressed air
3. Equip in the back accessory slot
4. Use controls to activate flight

**Jetpack Controls:**
- **Activation**: Configured keybind (check controls menu)
- **Vertical Movement**: Jump to ascend, sneak to descend
- **Horizontal Movement**: Standard WASD movement controls
- **Air Consumption**: Uses compressed air while active

**Tips:**
- Keep multiple jetpacks filled for extended flight
- Combine with elytra for maximum mobility
- Use in conjunction with Create trains for transportation

### Backpack Integration

**Sophisticated Backpacks:**
- Sophisticated Backpacks can be equipped in accessory slots
- Provides easy access to backpack inventory
- Can be upgraded with various enhancements

**Equipping Backpacks:**
1. Craft or obtain a Sophisticated Backpack
2. Equip in the back accessory slot
3. Access backpack inventory through the accessory slot
4. Upgrade with filters, auto-pickup, and more

**Backpack Features:**
- **Auto-Pickup**: Automatically collect items matching filters
- **Magnet Upgrades**: Pull items from nearby areas
- **Fluid Storage**: Store liquids in backpack tanks
- **Filtering**: Set up item filters for organization

## Slot Management

### Understanding Slot Types

Each accessory slot has specific item compatibility:

**Back Slot:**
- Elytra (from Elytra Slot mod)
- Create Backtanks (via Create: Curios integration)
- Create Jetpacks (via Create: Curios integration)
- Sophisticated Backpacks
- Other back-worn items

**Necklace Slot:**
- Necklaces from various mods
- Amulets and pendants
- Some utility items

**Ring Slots:**
- Multiple ring slots available
- Can equip different rings simultaneously
- Rings from various mods

**Belt Slot:**
- Utility belts
- Pouches and bags
- Some tool holders

### Managing Multiple Accessories

**Best Practices:**
1. **Prioritize Functionality**: Equip items you use most frequently
2. **Balance Slots**: Don't overload - some items may conflict
3. **Hot-Swapping**: Keep spare accessories in inventory for quick swaps
4. **Organization**: Use shulker boxes to store accessory collections

**Common Accessory Combinations:**
- **Exploration**: Elytra + Backpack + Utility Belt
- **Create Automation**: Backtank + Jetpack + Tool Belt
- **Combat**: Necklace + Rings + Backpack
- **Building**: Backpack + Tool Belt + Utility Items

### Accessory Inventory Management

**Organizing Accessories:**
- Keep frequently used accessories in hotbar slots
- Store backup accessories in ender chest
- Use shulker boxes for accessory organization
- Label containers for easy identification

**Quick Access Tips:**
- Set up keybinds for quick inventory access
- Use inventory sorting mods to organize accessories
- Create accessory storage areas in your base

## Advanced Tips & Tricks

### Optimizing Accessory Usage

**Efficiency Tips:**
- **Hot-Swap Setup**: Keep multiple accessory sets ready for different activities
- **Automated Refilling**: Use Create automation to refill backtanks automatically
- **Backup Equipment**: Always carry spare accessories for critical items
- **Slot Prioritization**: Equip most-used items in primary slots

### Troubleshooting

**Common Issues:**

**Accessory Won't Equip:**
- Check if item is compatible with slot type
- Verify mod compatibility
- Ensure item isn't damaged or broken
- Try relogging if slot appears bugged

**Accessory Not Working:**
- Check if item needs to be charged/filled (backtanks, jetpacks)
- Verify item is properly equipped (not just in inventory)
- Check mod configuration for slot requirements
- Ensure all required mods are installed

**Slot Conflicts:**
- Some items may conflict if equipped simultaneously
- Try equipping one at a time to identify conflicts
- Check mod documentation for known conflicts
- Some slots may have limited capacity

### Performance Considerations

**Optimization:**
- Accessory system has minimal performance impact
- Multiple accessories may slightly increase render load
- Curios API is optimized for performance
- Accessories mod is lightweight and efficient
