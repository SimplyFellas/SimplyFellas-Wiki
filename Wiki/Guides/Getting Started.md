# Getting Started

Welcome to SimplyFellas! This guide will help you take your first steps in the modpack and get you started on your journey.

---

## Overview

SimplyFellas is a carefully curated modpack that combines Create automation with extensive quality of life improvements, visual enhancements, and content additions. This guide covers everything you need to know to get started, from your first moments in the world to setting up your initial base.

---

## Prerequisites

Before starting, ensure you have:
- Successfully installed SimplyFellas (see [Wiki Homepage](Wiki-Homepage) for installation)
- Allocated at least 6-8GB of RAM
- Familiarity with basic Minecraft mechanics
- A new world created or an existing world to play in

---

## First Steps in the Modpack

### 1. Spawn and Initial Exploration

**When you first spawn:**

1. **Take a Look Around**: Notice the enhanced visuals and new structures
2. **Check Your Inventory**: You'll see JEI/EMI on the right side - this is your recipe guide
3. **Press `J`**: Open JourneyMap to see your location and set a waypoint
4. **Press `R` on Any Item**: See how to craft it (try it on wood or stone)

**Initial Goals:**
- Gather basic resources (wood, stone, food)
- Find a good location for your base
- Set a waypoint at your spawn or base location
- Familiarize yourself with the new UI elements

### 2. Essential Quality of Life Features

**Learn These First:**

**JEI/EMI (Recipe Guide):**
- **`R` Key**: Shows recipes for any item you're looking at
- **`U` Key**: Shows what an item is used for
- **Search Bar**: Type to find items quickly
- This is your most important tool - use it constantly!

**JourneyMap:**
- **`J` Key**: Opens full map
- **Right-click on Map**: Create waypoints
- **Minimap**: Appears in corner automatically
- Essential for navigation and not getting lost

**FTB Ultimine:**
- **Hold `~` Key**: While mining to mine 3x3 areas
- Great for clearing space and mining ores
- Saves tons of time

**Inventory Essentials:**
- **Middle-Click**: Automatically sorts any inventory
- **Various Keybinds**: Quick item transfer (check Controls menu)
- Makes inventory management much easier

### 3. Basic Resource Gathering

**Essential Resources to Gather:**

**Early Game:**
- **Wood**: Multiple types for different uses
- **Stone**: Cobblestone, stone, and variants
- **Food**: Hunt animals or find food sources
- **Coal/Charcoal**: For torches and fuel
- **Iron**: Essential for tools and Create

**Using JEI to Find Recipes:**
1. Press `R` on any item you want to craft
2. See all available recipes
3. Click the `+` button to transfer items to crafting grid
4. Craft the item

**Resource Gathering Tips:**
- Use Ultimine (`~` key) for faster mining
- Mark resource locations with JourneyMap waypoints
- Use backpacks (when you get them) for more storage
- Set up basic storage early

## Initial Base Setup

### Choosing a Location

**Good Base Locations Have:**
- **Flat Area**: Easier to build on
- **Resources Nearby**: Wood, stone, water
- **Water Source**: For Create waterwheels
- **Space to Expand**: Room for automation
- **Good Lighting**: Safe from mobs

**Location Tips:**
- Near a village (trading and resources)
- Close to different biomes (variety)
- Access to water (Create power)
- Not too far from spawn (convenience)

### Basic Base Structure

**Essential Buildings:**

**1. Storage Room:**
- Start with chests
- Organize by resource type
- Upgrade to storage networks later
- Keep it organized from the start

**2. Crafting Area:**
- Crafting table
- Furnace(s) for smelting
- Chest for materials
- Easy access to JEI

**3. Bedroom/Safe Area:**
- Bed for spawn point
- Safe from mobs
- Basic lighting
- Storage for personal items

**4. Future Automation Space:**
- Leave room for Create contraptions
- Plan for expansion
- Consider power generation area
- Space for farms and processing

### Base Setup Checklist

- [ ] Storage system (chests or better)
- [ ] Crafting area with easy JEI access
- [ ] Bedroom with bed
- [ ] Lighting to prevent mob spawns
- [ ] Basic farms (food source)
- [ ] Waypoint set at base location
- [ ] Organized resource storage
- [ ] Space planned for future expansion

## Key Quality of Life Features

### Inventory Management

**Inventory Essentials:**
- Middle-click to sort inventories
- Keybinds for quick item transfer
- Makes organization much easier
- Check Controls menu for all keybinds

**Trash Slot:**
- Trash slot in inventory
- Quickly dispose of unwanted items
- No need for lava or void blocks
- Very convenient

**Crafting Tweaks:**
- Buttons to rotate, balance, clear crafting grid
- Saves time when crafting
- Less manual item moving

### Information Display

**Jade (Block Information):**
- Look at blocks to see information
- Shows block names, inventory contents
- Entity information
- Very helpful for understanding mods

**AppleSkin:**
- See food saturation and hunger values
- Know exactly what food does
- Plan your food consumption
- Hover over food items to see stats

**Durability Tooltip:**
- See exact durability of tools
- Know when to repair
- Plan tool usage
- Hover over tools to see durability

### Navigation

**JourneyMap:**
- Full world map
- Minimap in corner
- Waypoint system
- Death point markers
- Essential for exploration

**Better Advancements:**
- Improved advancements menu
- Track your progress
- See what's available
- Better organization

## Common Beginner Mistakes

### Mistakes to Avoid

**1. Ignoring JEI/EMI:**
- Don't try to remember all recipes
- Use `R` and `U` constantly
- Search for items you need
- JEI is your best friend

**2. Not Using Waypoints:**
- Mark important locations
- Don't rely on memory
- Use JourneyMap waypoints
- Saves time and prevents getting lost

**3. Poor Organization:**
- Organize storage from the start
- Don't let chests get messy
- Use sorting features
- Plan for expansion

**4. Ignoring Create:**
- Create is the core of the pack
- Start learning it early
- Don't avoid automation
- It makes everything easier

**5. Not Reading Guides:**
- These guides exist to help
- Read before asking questions
- Follow progression paths
- Reference as needed
