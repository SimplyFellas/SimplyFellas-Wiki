# Exploration & Structures

Comprehensive guide to structure locations, exploration strategies, loot priorities, and dimension-specific information for SimplyFellas.

---

## Overview

SimplyFellas includes extensive structure mods that enhance world generation across all dimensions. This guide helps you find structures, prioritize exploration, use exploration tools effectively, and understand dimension-specific content.

---

## Prerequisites

Before starting this guide, you should:
- Have basic survival gear
- Understand JourneyMap basics
- Be ready to explore
- Have food and supplies

---

## Key Concepts

### Structure Types

**Vanilla Structures (Enhanced):**
- Dungeons, Mineshafts, Strongholds
- Temples, Monuments, Fortresses
- All enhanced by YUNG's Better Structures

**New Structures:**
- Moog's Structures (130+ new structures)
- Explorify structures
- Create-themed structures
- Dimension-specific structures

### Exploration Tools

**JourneyMap:**
- Full world map
- Waypoint system
- Structure markers
- Navigation aid

**Jade:**
- Block information
- Structure identification
- Entity information
- Helpful for exploration

## Structure Location Strategies

### Finding Structures

**Overworld Structures:**
- Explore different biomes
- Use JourneyMap to locate
- Look for structure markers
- Check common spawn locations

**Nether Structures:**
- Enhanced fortresses
- New Nether structures
- Graveyards and altars
- Explore Nether biomes

**End Structures:**
- Enhanced End cities
- New End structures
- Pirate ships and islands
- Explore outer End islands

### Exploration Tips

**Efficient Exploration:**
- Use elytra for fast travel
- Set waypoints at structures
- Mark important locations
- Plan exploration routes

**Preparation:**
- Bring food and supplies
- Have backup gear
- Use JourneyMap waypoints
- Prepare for combat
