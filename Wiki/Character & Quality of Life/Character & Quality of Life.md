# Character & Quality of Life

The SimplyFellas modpack includes an extensive collection of mods that enhance both your character appearance and provide countless quality of life improvements. These mods streamline gameplay, improve information display, enhance inventory management, and add visual polish to your Minecraft experience.

## About

SimplyFellas features nearly 40 quality of life mods that work together to create a polished, user-friendly experience. These mods cover everything from inventory management and crafting improvements to UI enhancements, character customization, and gameplay tweaks. Whether you're looking for better item organization, improved information display, or cosmetic enhancements, this modpack has you covered.

---

## Getting Started

### Essential QoL Features to Learn First

When starting with SimplyFellas, these quality of life features will have the biggest immediate impact:

1. **JEI/EMI** - Recipe viewing and item searching (Press `R` for recipes, `U` for uses)
2. **FTB Ultimine** - Fast mining (Hold `~` key while mining)
3. **JourneyMap** - Map and waypoints (Press `J` to open map)
4. **Jade** - Block information display (Look at blocks to see info)
5. **Inventory Essentials** - Quick item sorting (Middle-click to sort)

### Quick Setup Guide

**First Steps:**
1. Open your inventory and familiarize yourself with JEI/EMI on the right side
2. Press `R` on any item to see its recipes
3. Press `J` to open JourneyMap and set your first waypoint
4. Try holding `~` while mining to use Ultimine
5. Middle-click in any inventory to automatically sort items

**Keybind Configuration:**
- Press `ESC` → `Options` → `Controls` to configure all keybinds
- Search for mod names to find specific mod keybinds
- Many mods have configurable keybinds for customization

## Mining & Tools

### FTB Ultimine

The classic ultimine mod that allows you to mine up to 64 of the same block type using the tilde key (~). This dramatically speeds up mining operations, allowing you to clear large areas efficiently. Simply hold the tilde key while mining and you'll mine a 3x3 area (or larger, depending on configuration) of the same block type.

#### How to Use Ultimine

**Basic Usage:**
1. **Hold the Tilde Key (`~`)**: Default keybind is the tilde key (usually above Tab)
2. **Mine a Block**: While holding `~`, mine any block
3. **Area Mining**: All connected blocks of the same type will be mined
4. **Release to Stop**: Release the key to stop area mining

**Mining Patterns:**
- **3x3 Area**: Default mines a 3x3 area around the target block
- **Vein Mining**: Mines all connected blocks of the same type
- **Up to 64 Blocks**: Can mine up to 64 blocks in one operation
- **Smart Mining**: Only mines blocks you can reach

**Configuration:**
- Press `ESC` → `Options` → `Controls` → Search "Ultimine"
- Configure keybind to your preference
- Some launchers allow changing the mining pattern size
- Can be disabled for specific block types if needed

**Tips:**
- Works great for clearing large areas
- Perfect for mining ores in veins
- Excellent for clearing stone and dirt
- Be careful around valuable blocks - it mines everything connected!
- Works with Create blocks when using Create: Ultimine mod

**Best Practices:**
- Use for bulk mining operations
- Great for clearing space for builds
- Perfect for mining ores efficiently
- Watch your tool durability - it uses durability for all blocks mined
- Use Mending enchantment to keep tools repaired

## UI & Information Mods

### Just Enough Items (JEI)

The classic inventory recipe mod that you know and love for making use of your modpacks. JEI provides comprehensive recipe viewing, item searching, and crafting assistance.

#### JEI Basics

**Opening JEI:**
- JEI appears on the right side of your inventory screen
- Shows all available items in the modpack
- Searchable and filterable item list
- Always accessible when inventory is open

**Key Features:**
- **Recipe Viewing**: Press `R` on any item to see how to craft it
- **Usage Viewing**: Press `U` on any item to see what it's used for
- **Item Search**: Type in the search bar to find items
- **Bookmarking**: Click the `+` button to bookmark items
- **Recipe Transfer**: Click the `+` button in recipe view to transfer items to crafting grid

**Essential Keybinds:**
- **`R`**: Show recipes for selected item
- **`U`**: Show uses for selected item
- **`Ctrl+F`**: Focus search bar
- **`Ctrl+O`**: Toggle JEI overlay
- **`Ctrl+B`**: Toggle bookmark overlay

**Search Functions:**
- Type item name to search
- Use `@modname` to filter by mod (e.g., `@create`)
- Use `#tag` to filter by tag (e.g., `#forge:ingots`)
- Use `$text` to search tooltips
- Combine filters for advanced searches

**Recipe Viewing:**
- Click `R` on any item to see all recipes
- Multiple recipes show as tabs
- Click recipe to see detailed ingredients
- Hover over ingredients to see their recipes
- Use `+` button to transfer items to crafting grid

**Bookmarking:**
- Click `+` on any item to bookmark it
- Bookmarked items appear in separate list
- Great for tracking items you need to craft
- Access bookmarks via keybind or button

**Tips:**
- Use search to quickly find items
- Bookmark items you're working towards
- Use `R` and `U` to understand item relationships
- Filter by mod to see what each mod adds
- Use recipe transfer to speed up crafting

### EMI (Eternal Material Inventory)

A premium enhancement for Just Enough Items that provides way more keybinds, configurations, and a much cleaner user interface. EMI offers advanced filtering, bookmarking, and recipe viewing capabilities.

#### EMI Features

**Enhanced Interface:**
- Cleaner, more modern interface than JEI
- Better organization and layout
- More intuitive controls
- Enhanced visual design

**Advanced Features:**
- **Better Search**: More powerful search with advanced filters
- **Enhanced Bookmarks**: Improved bookmarking system
- **Recipe Favorites**: Mark favorite recipes for quick access
- **Crafting History**: See recently viewed recipes
- **Better Integration**: Works seamlessly with other mods

**Keybinds:**
- Most JEI keybinds work the same (`R`, `U`, etc.)
- Additional EMI-specific keybinds available
- Check Controls menu for full keybind list
- Highly configurable

**Search Enhancements:**
- More powerful search algorithms
- Better autocomplete
- Advanced filter options
- Improved search results

**Tips:**
- EMI works alongside JEI - both are active
- Use EMI for enhanced features
- JEI compatibility maintained
- Best of both worlds

Jade Additions integrates additional mods' information into Jade.

### JourneyMap

Adds a comprehensive map, minimap, and waypoint marker system to the modpack. Perfect for navigation and exploration tracking.

#### JourneyMap Basics

**Opening the Map:**
- **Full Map**: Press `J` (default) to open full-screen map
- **Minimap**: Appears in corner of screen automatically
- **Waypoints**: Right-click on map to create waypoints
- **Navigation**: Use map to plan routes and mark locations

**Key Features:**
- **Full World Map**: Complete map of explored areas
- **Minimap**: Real-time minimap in corner
- **Waypoints**: Mark important locations
- **Death Points**: Automatically marks death locations
- **Biome Display**: Shows biomes on map
- **Structure Markers**: Marks discovered structures

**Essential Keybinds:**
- **`J`**: Open/close full map
- **`B`**: Open waypoint manager
- **`N`**: Create new waypoint at current location
- **`M`**: Toggle minimap
- **`Ctrl+M`**: Minimap options

**Creating Waypoints:**
1. Press `J` to open map
2. Right-click on desired location
3. Enter waypoint name
4. Choose icon and color
5. Waypoint appears on map and minimap

**Waypoint Management:**
- Press `B` to open waypoint manager
- Edit, delete, or organize waypoints
- Set waypoint visibility
- Create waypoint categories
- Share waypoints with other players (multiplayer)

**Map Features:**
- **Zoom**: Scroll to zoom in/out
- **Pan**: Click and drag to move map
- **Search**: Search for biomes, structures, waypoints
- **Layers**: Toggle different map layers
- **Export**: Export map images

**Minimap Features:**
- **Position**: Configurable position on screen
- **Size**: Adjustable minimap size
- **Zoom**: Independent zoom level
- **Radar**: Show entities on minimap
- **Cave Mode**: See underground areas

**Tips:**
- Mark your base location immediately
- Create waypoints for important structures
- Use death points to recover items
- Explore to fill in map
- Use minimap for navigation while exploring
- Export maps for sharing or planning

## World & Mechanics

### Amendments

Adds cauldron potion mixing, more animations, more block combinations, and much more to enhance vanilla mechanics.

### Supplementaries

Adds various Minecraft-like objects that fill in the gaps that vanilla Minecraft has, enhancing world interaction and decoration.

### Anvil Never Too Expensive

Removes the anvil "too expensive" limitation, allowing you to repair and combine items indefinitely.

### Flat EXP Costs

Enchanting and using the anvil costs the actual amount of XP it should cost rather than the number of levels, making XP management more predictable.

### Nether Portal Fix

Fixes nether portals to use the correct coordinates, preventing portal linking issues.

### Crying Portals

Allows you to make nether portals out of crying obsidian! Adds visual variety to portal construction.

### Useful Snow Golems

Makes snow golems do a very light amount of damage to make them actually useful for defense and mob farms.

### No Feather Trample

Prevents you from trampling farmland when wearing feather falling boots, protecting your crops.

### Followers Teleport Too

When you use a command to teleport somewhere, your followers (tamed animals, villagers, etc.) will be brought with you automatically.

### Liberty's Villagers Revived

This mod has been configured to:
- Change villagers' workstation range to 3 blocks (improves AI)
- Allow clerics to throw potions at players within 10 blocks
- Allow you to feed villagers various food items to heal them

## Troubleshooting

### Common Issues

**JEI/EMI Not Showing:**
- Check if mods are installed correctly
- Try pressing `Ctrl+O` to toggle overlay
- Restart game if issue persists
- Check for mod conflicts

**JourneyMap Not Working:**
- Press `J` to open map manually
- Check if minimap is enabled
- Verify mod installation
- Check keybind configuration

**Ultimine Not Working:**
- Check keybind configuration
- Ensure you're holding the key while mining
- Verify tool can mine the block type
- Some blocks may be disabled

**Inventory Sorting Issues:**
- Ensure Inventory Essentials is installed
- Check keybind configuration
- Try middle-clicking in inventory
- Some inventories may not support sorting

### Getting Help

- Check mod documentation on CurseForge
- Visit SimplyFellas Discord/Forums
- Check [Getting Started Guide](Getting-Started)
- Browse other wiki sections for related information
